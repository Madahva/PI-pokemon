# Pokedex-SPA
This is a React <strong>SPA</strong> and <strong>CRUD</strong> application with certain features such as: Pagination, Filter, Sort, Fetching Data, Responsive, etc.
    
## Built with
- React
- Redux
- Semantic HTML5 markup
- CSS Flexbox
- Mobile-first workflow
- API
- Figma
- Node.js
- Express
- Sequelize
- Postgres
- SQL
