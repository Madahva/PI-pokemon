# Pokedex-SPA
This is a React SPA and CRUD application with certain features such as: Filter, Sort ,Fetching Data, Responsive, etc.
    
## Built with
- React
- Redux
- Semantic HTML5 markup
- CSS Flexbox
- Mobile-first workflow
- API
- Figma
- Node.js
- Express
- Sequelize
- Postgres
- SQL
